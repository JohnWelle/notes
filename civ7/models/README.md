## Model Images

Use this folder to add images used to capture evidence as part of running model test cases.

Use a file naming convention that references the file name of the model that is being tested, e.g.

```
models/
├── great-wall
    great-wall.md
    └── images/
        ├── great-wall-test-1.png
        ├── great-wall-test-2.png
        └── great-wall-ugly-lattice.png
└── rehydrate-context/

```
